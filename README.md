# 건설人 (건설인) - 건설 인력 매칭 플랫폼

건설 현장에서 필요한 기술자와 기업을 연결하는 AI 기반 매칭 플랫폼입니다.

## 기술 스택

| 구성요소 | 기술 |
|----------|------|
| 프레임워크 | Next.js 15 (App Router) |
| 언어 | TypeScript |
| 스타일링 | Tailwind CSS |
| 데이터베이스 | Supabase (PostgreSQL) |
| 인증 | Supabase Auth (이메일/비밀번호, 카카오, 네이버) |
| 실시간 | Supabase Realtime (채팅) |
| AI 엔진 | OpenRouter (Gemini 3.1 Flash Lite) |
| 배포 | Vercel |

## 빠른 시작 가이드

### 1. 회원가입

1. 홈페이지에서 **회원가입** 클릭
2. 이메일/비밀번호 입력 또는 소셜 로그인 (카카오, 네이버)
3. **역할 선택**: 기술자(Worker) 또는 기업(Company)
4. 프로필 작성 (온보딩 과정에서 안내)

### 2. 기술자 사용법

#### 프로필 설정
- **설정 > 프로필**: 전문 분야, 경력, 보유 기술, 희망 시급 입력
- 프로필 완성도 100%가 되어야 공고 지원 가능

#### 공고 검색 및 지원
1. **공고 목록** 페이지에서 지역, 직종, 급여 조건으로 검색
2. 관심있는 공고 클릭 → 상세 정보 확인
3. **지원하기** 클릭 → 자기소개/각오 작성 후 제출

#### AI 매칭 (자동 추천)
1. **대시보드**에서 "AI 추천 공고" 확인
2. 내 프로필(전문 분야, 위치, 경력)에 맞는 공고 자동 추천
3. **설정 > AI 매칭**에서 매칭 점수 기준, 최대 추천 수 조절 가능

#### 자동 지원
1. **설정 > 자동 지원**에서 기능 활성화
2. 자동 지원 조건 설정 (직종, 지역, 최소 급여 등)
3. AI가 15분 간격으로 조건에 맞는 공고에 맞춤 메시지 작성 후 자동 지원

#### 채팅
- **채팅** 페이지에서 기업 담당자와 실시간 대화
- 공고 상세 페이지에서 **채팅 문의** 버튼으로 바로 시작

### 3. 기업 사용법

#### 공고 등록
1. **공고 등록** 페이지에서 공고 작성
2. 필수 정보: 공고 제목, 근무지, 급여, 직종, 상세 설명
3. 선택 정보: 자격 요건, 복리후생, 마감일

#### 지원자 관리
1. 공고 상세 > **지원자 보기**
2. 지원자 목록에서 프로필 확인
3. **AI 분석** 버튼으로 전체 지원자 적합도 일괄 평가
   - 6개 항목: 직종 일치(30), 기술 매칭(25), 경력(15), 평점(15), 위치(10), 급여(5)
   - 등급: A(80+), B(60-79), C(40-59), D(0-39)

#### AI 챗봇 설정 (기업 전용)
1. **설정 > AI 챗봇**에서 활성화
2. 챗봇 톤 설정: 격식체 / 존대말 / 간결체
3. 운영 시간: 항상 / 업무 외 시간 / 커스텀
4. **FAQ 관리**: 자주 묻는 질문과 답변 등록
5. 기술자가 채팅으로 문의하면 AI가 자동 응답

#### 계약 및 결제
1. 적합한 기술자 선택 후 **계약 생성**
2. 일급, 근무일수, 시작/종료일 설정
3. 에스크로 결제 → 작업 완료 확인 → 정산

### 4. 공통 기능

#### 리뷰 시스템
- 계약 완료 후 상대방 리뷰 작성 가능
- 별점(1-5) + 카테고리별 평가 + 코멘트

#### 설정
- **알림**: 푸시, 이메일, 채팅 알림 on/off
- **개인정보**: 프로필 공개, 위치 공유 설정
- **AI 설정**: 매칭, 자동 지원 세부 조건 조절

#### AI 도우미
- 화면 오른쪽 하단 **💬 버튼**을 클릭하면 AI 도우미가 열립니다
- 플랫폼 사용법, 기능 설명 등 궁금한 점을 자유롭게 질문하세요

## 로컬 개발

```bash
# 의존성 설치
npm install

# 환경변수 설정
cp .env.example .env.local
```

### 필수 환경변수

| 변수 | 설명 |
|------|------|
| `NEXT_PUBLIC_SUPABASE_URL` | Supabase 프로젝트 URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Supabase anon key |
| `SUPABASE_SERVICE_ROLE_KEY` | Supabase service role key (서버 전용) |
| `OPENROUTER_API_KEY` | OpenRouter API 키 (AI 기능) |

```bash
# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 데이터베이스

Supabase 마이그레이션 파일은 `supabase/migrations/` 디렉토리에 있습니다.

```bash
# Supabase 로컬 시작
supabase start

# 마이그레이션 적용
supabase db push

# 타입 생성
supabase gen types typescript --local > lib/supabase/types.ts

# 데모 데이터 시드
supabase db seed
```
